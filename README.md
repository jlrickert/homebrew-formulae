# homebrew-formulae

Homebrew tap for CLI tools by [@jlrickert](https://github.com/jlrickert).

## Install

```bash
brew tap jlrickert/formulae
brew install tapper    # KEG management CLI (installs `tap` binary)
brew install keg       # Project-local KEG tool (installs `keg` binary)
```

Or install directly:

```bash
brew install jlrickert/formulae/tapper
brew install jlrickert/formulae/keg
```

## Available Formulas

| Formula  | Binary | Description                            |
| -------- | ------ | -------------------------------------- |
| tapper   | `tap`  | Full CLI for managing KEG knowledge graphs |
| keg      | `keg`  | Project-local KEG management tool      |

## How it works

Formulas are auto-generated by [GoReleaser](https://goreleaser.com/) on each
release of [tapper](https://github.com/jlrickert/tapper). Do not edit formula
files manually.
